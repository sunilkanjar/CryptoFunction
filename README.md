# HashCalculator

Pure java implementaion of **Hashing** and **CRC** Listed in [This Page](https://en.wikipedia.org/wiki/List_of_hash_functions)

  * CRC
    * BSD checksum
    * Checksum
    * CRC-16
    * CRC-32
    * CRC-64
    * SYSV checksum
  * Checksum
    * Sum8
    * Sum16
    * Sum24
    * Sum32
    * Fletcher-4
    * Fletcher-8
    * Fletcher-16
    * Fletcher-32
    * Adler-32
    * XOR8/LRC/Longitudinal redundancy check
  * Non-cryptographic hash functions
    * Pearson
    * Fowler–Noll–Vo 0 32
    * Fowler–Noll–Vo 1 32
    * Fowler–Noll–Vo 1a 32
    * Fowler–Noll–Vo 0 64
    * Fowler–Noll–Vo 1 64
    * Fowler–Noll–Vo 1a 64
